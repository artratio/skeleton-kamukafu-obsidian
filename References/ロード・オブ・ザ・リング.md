# ロード・オブ・ザ・リング

## Overview

ピーター・ジャクソン監督による映画三部作（2001-2003年）。[[指輪物語]] の映像化作品。アカデミー賞計17部門受賞。

トールキンがテキストだけで描いた中つ国の壮大な風景、ゴラムの表情、ヘルム峡谷の合戦 — すべてはプレーンテキストから生まれた。

小説が映画になったのだ。ならば、あなたが書いたテキストが PowerPoint になるくらい、造作もないことだ。
