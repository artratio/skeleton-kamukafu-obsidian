# プレーンテキスト

プレーンテキストとは人間が読めるテキストのこと。

メモ帳のリッチテキストもそうだし、Markdown 記法もそうだし、コードもプレーンテキスト。HTML と CSS も。`.txt`、`.md`、`.html`、`.css`、`.py`、`.json` — コードや構造化データも含んだ総称。

[[Think to Text Protocol]] で「テキスト」と呼んでいるのは、この広義のプレーンテキストのこと。特定のアプリ経由でなくても読める文章。

[[File over app]] の思想が成り立つのも、プレーンテキストが最も長寿で汎用的なフォーマットだからだ。
