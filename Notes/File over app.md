# File over app

[[StephAngo]]（kepano）が提唱する哲学。

> *File over app is a philosophy: if you want to create digital artifacts that last, they must be files you can control, in formats that are easy to retrieve and read. Use tools that give you this freedom.*

## 要点

- アプリは消える。ファイルは残る
- プレーンテキストは最も長寿なフォーマット。Markdown はその上に構築された軽量な構造化記法
- データの所有権はユーザーにあるべき

## なぜ重要か

Notion は素晴らしいが、Notion が消えたらデータはどうなる？ Evernote は素晴らしかったが、多くの人がデータの移行に苦労した。

Obsidian の Vault はただの Markdown ファイルの集合体。Obsidian がなくなっても、任意のテキストエディタで読み書きできる。

AI にとっても、[[プレーンテキスト]] は最も扱いやすいインターフェース。[[Think to Text Protocol]] との親和性が高い理由はここにある。
